# LUMINA - Design System

## 1. Typography
**Primary Font (Headings/Brand):** `Cinzel` (Serif)
*   Usage: Logo, Section Headers, Hero Titles.
*   Weights: 400 (Regular), 600 (Bold).

**Secondary Font (Body/UI):** `Noto Sans KR` (Sans-serif)
*   Usage: Body text, Buttons, Inputs, Navigation.
*   Weights: 300 (Light), 400 (Regular), 500 (Medium), 700 (Bold).

**Accent Font:** `Noto Serif KR` (Serif)
*   Usage: Korean headings, emotional text.

## 2. Color Palette

### Primary Colors
*   **Primary Black:** `#1a1a1a` (Text, Footer, Primary Buttons)
*   **White:** `#ffffff` (Backgrounds, Cards)

### Secondary & Accents
*   **Gold (Accent):** `#d4af37` (Highlights, Links, Special Badges)
*   **Secondary BG:** `#f8f8f8` (Section backgrounds, Inputs)
*   **Border Gray:** `#e5e7eb` (Dividers, Card borders)

### Admin Palette
*   **Sidebar:** `#1e293b` (Slate 800)
*   **Order Status (Blue):** `#eff6ff` (bg), `#3b82f6` (text)
*   **Claim Status (Pink):** `#fff1f2` (bg), `#f43f5e` (text)

## 3. UI Components

### Buttons
*   **Style:** Sharp corners (rounded-none or rounded-sm), uppercase text, wide tracking.
*   **Primary:** Dark background, White text.
*   **Secondary:** White background, Dark border.

### Product Cards
*   **Image:** 4:5 Aspect Ratio (Portrait) or 1:1 (Square for sliders).
*   **Interactivity:** Hover zoom (removed per request), quick-add button.

### Layouts
*   **Container:** `max-w-6xl`, centered.
*   **Grid:** Responsive (1 col mobile, 2 col tablet, 3-4 col desktop).
*   **Admin:** Sidebar + Top Header + Content Area.

## 4. Icons
*   **Library:** `lucide-react`
*   **Style:** Minimalist line icons (Stroke width 1.5 or 2).
