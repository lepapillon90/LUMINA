# AI Studio 앱 실행 및 배포

이 문서는 앱을 로컬에서 실행하는 데 필요한 모든 정보를 포함합니다.

AI Studio에서 앱 보기: https://ai.studio/apps/drive/16e4x0dTv-d0wHQMwbOn50tosNzdDuVX2

## 로컬에서 실행하기 (Run Locally)

**필수 조건:** Node.js

1. 의존성 설치:
   `npm install`
2. [.env.local](.env.local) 파일에 `GEMINI_API_KEY`를 설정하여 Gemini API 키 입력
3. 앱 실행:
   `npm run dev`
