# Admin Dashboard Future Roadmap

관리자 대시보드는 쇼핑몰 운영의 심장입니다. 효율적인 운영, 데이터 기반의 의사결정, 고객 만족도 관리를 위한 기능 개선 로드맵입니다.

## 1단계: 주문 및 배송 운영 효율화 (Operations)
> **목표**: 주문 처리 속도 향상 및 오배송 제로화
- [x] **송장 자동 출력**: 택배사 API 연동을 통해 관리자 페이지에서 운송장 바로 출력.
- [x] **주문 상태 일괄 변경**: 여러 주문을 선택하여 '배송 준비 중', '배송 중'으로 한 번에 변경.
- [x] **CS 통합 관리**: 상품 문의, 1:1 문의, 반품 요청을 한 화면에서 처리하고 답변 템플릿 활용.
- [x] **재고 알림 설정**: 안전 재고 이하로 떨어진 상품에 대해 관리자에게 알림 발송.

## 2단계: 상품 및 전시 관리 (Merchandising)
> **목표**: 매력적인 상품 진열 및 프로모션 관리
- [x] **진열 관리 (Drag & Drop)**: 홈 화면의 추천 상품이나 카테고리별 상품 순서를 드래그 앤 드롭으로 변경.
- [x] **배너/팝업 관리**: 개발자 도움 없이 메인 배너 이미지와 링크, 팝업창을 기간 설정하여 등록.
- [x] **기획전 생성기**: 특정 상품들을 묶어 별도의 랜딩 페이지(기획전)를 생성하는 빌더.

## 3단계: 데이터 분석 및 인사이트 (Analytics)
> **목표**: 감이 아닌 데이터에 기반한 의사결정
- [x] **실시간 대시보드**: 현재 방문자 수, 실시간 매출, 인기 상품을 라이브로 모니터링.
- [x] **매출 리포트**: 일별/주별/월별 매출 추이, 객단가, 재구매율 등을 시각화된 차트로 제공.
- [x] **고객 분석 (CRM)**: VIP 고객, 이탈 위험 고객, 잠재 고객군을 분류하여 마케팅 타겟 추출.
- [x] **유입 경로 분석**: 고객이 인스타그램, 검색, 광고 중 어디를 통해 들어왔는지 추적.

## 4단계: 시스템 및 권한 관리 (System)
> **목표**: 운영 보안 강화 및 협업 효율성 증대
- [x] **부운영자 권한 관리**: MD, CS, 배송 담당자별로 접근 가능한 메뉴 권한 세분화.
- [x] **활동 로그 (Audit Log)**: 누가 언제 어떤 주문을 수정했는지, 상품을 삭제했는지 기록 추적.
- [x] **모바일 어드민**: 외부에서도 급한 주문 처리나 매출 확인이 가능한 모바일 전용 관리자 모드.
