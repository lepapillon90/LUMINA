# Shopping Bag (Cart) Page Future Roadmap

장바구니 페이지는 구매 결정의 마지막 단계입니다. 결제 이탈(Cart Abandonment)을 최소화하고 객단가(AOV)를 높이는 전략적인 기능 개선 로드맵입니다.

## 1단계: 구매 전환 유도 (Conversion Optimization)
> **목표**: 장바구니에 담긴 상품을 실제 결제로 연결하고 객단가 상승 유도
- [x] **무료 배송 게이지**: "5,000원 더 담으면 무료 배송!"과 같이 남은 금액을 시각적인 바(Bar) 형태로 표시.
- [x] **크로스 셀링 (Cross-selling)**: 장바구니에 담은 상품과 어울리는 저렴한 액세서리(예: 쥬얼리 클리너, 선물 포장) 추천.
- [x] **쿠폰 자동 적용**: 사용 가능한 쿠폰 중 가장 혜택이 큰 쿠폰을 자동으로 찾아 적용해주는 기능.

## 2단계: 사용자 편의성 (User Convenience)
> **목표**: 결제 과정의 장애물을 제거하고 유연한 쇼핑 경험 제공
- [x] **나중에 구매하기 (Save for Later)**: 지금 당장 사지 않을 상품을 장바구니에서 삭제하지 않고 별도 리스트로 이동.
- [ ] **옵션 변경 기능**: 장바구니 내에서 상품 상세 페이지로 이동하지 않고 수량, 사이즈, 색상 옵션 즉시 변경.
- [ ] **비회원 주문 조회**: 로그인 없이도 주문 번호와 이메일로 배송 상태를 조회할 수 있는 기능.
- [x] **선물하기 옵션**: "선물 포장" 체크박스 추가 및 메시지 카드 작성 기능.

## 3단계: 신뢰도 및 심리적 안정감 (Trust & Security)
> **목표**: 결제 직전의 불안감을 해소
- [x] **결제 수단 아이콘 노출**: 지원하는 카드사, 간편결제(카카오페이, 토스 등) 로고를 명시하여 신뢰도 상승.
- [x] **안심 문구 배치**: "100% 정품 보장", "무료 반품 가능", "보안 서버 결제" 등의 마이크로 카피 추가.
- [x] **예상 배송일 안내**: 오늘 주문 시 언제 도착하는지 구체적인 날짜(예: "12/1(월) 도착 예정") 표시.

## 4단계: 데이터 및 리타겟팅 (Retention)
> **목표**: 이탈한 고객을 다시 불러오기
- [ ] **장바구니 리마인더**: 장바구니에 상품을 담고 이탈한 회원에게 푸시 알림이나 이메일 발송 (재고 부족 임박 알림과 결합).
- [ ] **영구 장바구니 (Persistent Cart)**: 모바일에서 담은 상품을 PC에서도 그대로 볼 수 있도록 기기 간 동기화.
