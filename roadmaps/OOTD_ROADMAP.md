# OOTD (Outfit Of The Day) Future Roadmap

현재의 OOTD 페이지는 기본적인 커뮤니티 기능을 갖추고 있습니다. 향후 사용자 참여를 높이고 매출로 연결하기 위한 단계별 개선 플랜을 제안합니다.

## 1단계: 데이터 영속성 및 백엔드 통합 (Foundation)
> **목표**: 새로고침 시 데이터가 사라지는 문제 해결 및 실제 사용자 데이터 관리
- [x] **Firebase Firestore 연동**: 게시물, 댓글, 좋아요 데이터를 DB에 저장.
- [x] **Firebase Storage 연동**: 사용자가 업로드한 이미지를 클라우드 스토리지에 저장.
- [x] **사용자 인증 연동**: 실제 로그인한 사용자 정보(프로필, ID)를 게시물 작성자 및 댓글 작성자로 연결.

## 2단계: 소셜 기능 강화 (Engagement)
> **목표**: 사용자 간 상호작용 증대 및 체류 시간 확보
- [x] **대댓글 기능**: 댓글에 답글을 달 수 있는 계층형 댓글 구조 도입.
- [x] **팔로우 시스템**: 마음에 드는 스타일의 사용자를 팔로우하고 피드에서 모아보기.
- [x] **공유하기**: 게시물을 카카오톡, 인스타그램 등으로 공유하는 기능.
- [x] **북마크/저장**: 나중에 참고하고 싶은 스타일을 '마이페이지'에 저장.

## 3단계: AI & 검색 경험 혁신 (Discovery)
> **목표**: 원하는 스타일을 쉽게 찾고, AI를 통한 자동화
- [x] **AI 자동 태깅 (Gemini)**: 사진 업로드 시 Gemini가 의상 스타일(예: 미니멀, 빈티지)과 색상을 분석해 자동 태그 생성.
- [x] **유사 스타일 추천**: 현재 보고 있는 OOTD와 비슷한 분위기의 다른 게시물 추천.
- [x] **고급 필터링**: 키, 체형, 스타일(캐주얼/포멀), 계절별 필터링 기능.

## 4단계: 커머스 연계 강화 (Conversion)
> **목표**: OOTD 콘텐츠를 실제 구매로 전환
- [x] **Shop the Look**: 게시물에 태그된 상품을 한 번에 장바구니에 담는 "모두 담기" 기능.
- [x] **유저 리워드**: 내 OOTD를 통해 상품 구매가 발생하면 포인트 적립 (앰배서더 프로그램).
- [x] **베스트 리뷰어 선정**: 주간/월간 좋아요 1위 선정 및 배지 부여.

## 5단계: 성능 및 UX 최적화 (Optimization)
- [x] **무한 스크롤 (Infinite Scroll)**: 더 많은 게시물을 끊김 없이 탐색하도록 개선.
- [x] **이미지 최적화**: 다양한 디바이스 해상도에 맞는 최적화된 이미지 서빙 (WebP 등).
- [x] **Masonry 레이아웃**: 핀터레스트 스타일의 동적 그리드 레이아웃 적용으로 시각적 재미 부여.
