# Shop Page Future Roadmap

쇼핑 페이지(상품 목록 및 상세)는 매출과 직결되는 핵심 영역입니다. 고객이 원하는 상품을 쉽고 빠르게 찾고, 구매 결정을 내릴 수 있도록 돕는 기능 개선 로드맵입니다.

## 1단계: 검색 및 필터링 고도화 (Discovery)
> **목표**: 고객이 원하는 상품을 빠르고 정확하게 찾을 수 있도록 지원
- [x] **상세 필터링**: 가격대(슬라이더), 소재(골드/실버), 색상, 사이즈별 필터 추가.
- [x] **스마트 검색**: 검색어 자동 완성, 오타 교정, 추천 검색어 제공.
- [x] **정렬 옵션 다양화**: 인기순, 판매량순, 리뷰 많은순, 높은 할인율순 정렬 추가.

## 2단계: 사용자 경험 개선 (UX Enhancement)
> **목표**: 쇼핑 과정에서의 피로도를 줄이고 즐거움 제공
- [x] **퀵뷰(Quick View)**: 상품 목록에서 상세 페이지 이동 없이 정보 확인 가능한 모달 구현.
- [x] **위시리스트 (찜하기)**: 마음에 드는 상품을 저장하고 나중에 모아보는 기능 (하트 버튼).
- [x] **품절 임박 알림**: 재고가 적은 상품에 "품절 임박" 배지 표시로 구매 유도.

## 3단계: 상품 상세 및 신뢰도 강화 (Conversion)
> **목표**: 상세 페이지에서의 이탈을 막고 구매 확신 부여
- [x] **연관 상품 추천**: "이 상품과 함께 많이 본 상품" 또는 "함께 코디하기 좋은 아이템" 추천 알고리즘.
- [x] **상세 사이즈 가이드**: 반지 호수 측정법 등 카테고리별 맞춤 사이즈 가이드 팝업. 
- [x] **재입고 알림 신청**: 품절 상품에 대해 재입고 시 알림톡/이메일 발송 신청 기능.

## 4단계: 기술 및 관리 효율화 (Infrastructure)
> **목표**: 데이터 기반의 운영 및 성능 최적화
- [x] **실시간 재고 연동**: 주문 발생 시 실시간으로 재고 차감 및 품절 처리 (Firestore Transaction).
- [x] **SEO 구조화 데이터**: 구글 등 검색 엔진에 상품 가격, 재고 상태, 평점 등이 노출되도록 Schema.org 마크업 적용.
